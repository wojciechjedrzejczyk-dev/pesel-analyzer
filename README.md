PESEL Analyzer

Prosty program konsolowy napisany w Pythonie służący do analizy numeru PESEL.


Funkcje:
* sprawdzanie poprawności długości numeru PESEL,
* odczytywanie daty urodzenia,
* określanie płci na podstawie numeru PESEL,
* wyświetlanie informacji o użytkowniku.


Technologie:
* Python


Jak uruchomić program?
1. Pobierz projekt
2. Uruchom plik `.py` w interpreterze Python


Przykładowe działanie:
Pesel poprawny!
Mężczyzna urodzony/a 27-go listopada 2004 roku
